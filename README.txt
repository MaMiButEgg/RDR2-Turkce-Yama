RDR2 Türkçe Yama Kullanım Kılavuzu
1. Gereksinimler

RDR2 orijinal veya crackli sürüm

ScriptHookV 1.0 veya üstü

dinput8.dll

Windows 7 / 10 / 11

2. Yama Yükleme

Programı açın.

“RDR2 Klasörü” alanına oyunun kurulu olduğu klasörü seçin.

“Yama Yükle” butonuna tıklayın.

Yükleme sırasında ilerleme çubuğu dolacak.

3. Dil Seçimi

Sağ üstteki “EN / TR” butonu ile arayüz dilini değiştirebilirsiniz.

4. Gereksinimler Butonu

Program penceresindeki “Gereksinimler” butonuna tıklayarak gerekli dosyalar ve bilgiler görüntülenebilir.

5. README Açıklaması

README dosyası, yama yüklemeden sonra otomatik olarak açılır ve kullanıcıya kılavuz niteliğinde bilgi verir.

6. Yama Kaldırma

Eğer yamanın kaldırılması gerekirse, aşağıdaki dosyalar silinmelidir:

rdr2-translator.asi

rdr2-translator.xml

Bu dosyaları RDR2’nin kurulu olduğu klasörden silmeniz yeterlidir.

Dosyalar silindikten sonra oyun eski haline dönecektir.

7. Diğer Notlar

Yama sadece oyunun kendi dosyalarını değiştirmez, ek dosyalar ekler.

Yükleme veya silme sırasında oyunu çalıştırmayın.

Herhangi bir sorun yaşarsanız, önce antivirus programınızın yama dosyalarını engellemediğinden emin olun.
